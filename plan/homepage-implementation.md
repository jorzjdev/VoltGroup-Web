# План: главная страница VoltGroup-Web

## Контекст и ограничения

- **Референс:** только главная страница [lumos-art.ru](https://lumos-art.ru/) — структура секций, типографика, поведение меню, карусели, табы.
- **Бренд:** «Вольтгрупп» (не «Люмос»).
- **Исключить:** функционал «Заказать звонок» (виджет обратного звонка / отдельный CTA заказа звонка). Телефон и email в блоке контактов и футере — оставить как справочную информацию.
- **Включить:** форма «Обсудить проект» (модальное окно + секция внизу страницы, как на референсе).
- **Контент:** временные заглушки; тексты/фото потом заменяются без перестройки вёрстки.
- **Стек:** Vite (выбор пользователя).
- **Форма:** только UI + клиентская валидация; отправку подключим позже.

## Архитектура проекта

**Почему Vite MPA (multi-page), а не SPA:** проще SEO и деплой статики; главная — полноценная landing-страница; будущие разделы (`/portfolio`, `/services`, `/about`) — отдельные HTML-точки входа без тяжёлого роутера.

### Целевая структура файлов

```
voltGroup/
  plan/
    homepage-implementation.md
  index.html
  pages/
    portfolio.html, services.html, about.html, contacts.html, blog.html, privacy.html
  src/
    styles/, scripts/, data/, assets/
  public/favicon.svg
  vite.config.js
  package.json
```

## Критерии готовности

- Главная соответствует референсу по структуре и UX.
- Брендинг «Вольтгрупп».
- Нет «Заказать звонок»; есть «Обсудить проект».
- Mobile + desktop.
- Якоря + stub-страницы разделов.
- Форма с валидацией и stub submit.
- `npm run build` без ошибок.
